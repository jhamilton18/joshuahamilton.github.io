The Impact of Algorithmic Content Delivery on Working Memory Capacity and Executive Attention

This study investigated how social media algorithms (content ranking, recommendation, and infinite scrolling) affect users' cognitive processes, specifically working memory capacity, executive attention, and cognitive load. Participants were exposed to different algorithm conditions and their cognitive performance was measured before and after exposure. While no significant differences were found in working memory or attention, infinite scrolling was associated with increased perceived cognitive load, consistent with previous research on information overload.

Reducing Human Error And Enhancing Security in EHR Systems

Healthcare systems face significant challenges in maintaining data security due to human error and outdated technology. EHRShield is a proposed digital application designed to enhance EHR security, user experience, and efficiency. By leveraging advanced machine learning techniques, EHRShield proactively detects and prevents potential security breaches, provides real-time feedback, and offers tailored training to healthcare professionals.

Virtual Reality Cybersecurity Experience

This project aims to create an immersive VR experience that simulates real-world cyber threats like hacking and phishing attacks. Users will learn cybersecurity skills by interacting with realistic virtual environments. The project will utilize A-Frame for the VR framework, 3D modeling tools like Blender, and game engines like Unity for complex simulations. It will track user performance to provide tailored training and consider accessibility and ethical implications to ensure a positive and safe learning experience.

Recursive HTTPS Requests and DDOS Attacks Simulation

This project aims to simulate real-world cyberattacks, such as recursive HTTPS requests and DDoS attacks, to gain insights into network behavior, identify vulnerabilities, and test security mitigation strategies. By combining technical simulation with human factors analysis, we can better understand the role of human behavior in cybersecurity incidents and develop more effective security solutions.

Intelligent Virtual Therapist: A personalized VR Therapy Experience

This project aims to create an immersive virtual reality (VR) experience that leverages the power of artificial intelligence (AI) to provide personalized cognitive behavioral therapy (CBT). The AI agent, implemented using a language model like ChatGPT, will interact with users in real-time, guiding them through CBT techniques, such as cognitive restructuring and exposure therapy. The VR environment, built using Unreal Engine, will provide a safe and controlled space for users to practice new skills and confront their fears. By combining the strengths of VR and AI, this project seeks to revolutionize mental health care by providing accessible and effective therapy
